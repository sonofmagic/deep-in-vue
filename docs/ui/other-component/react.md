# React 组件生态速览

这一页不是要你把 React 方案搬进 Vue，而是把它当成一个对照坐标。

React 组件生态最大的价值，在于它把“组件库工程化”这件事做得非常成熟。  
所以这里更值得看的，不是某个具体 API，而是它长期沉淀出来的工程习惯。

## 值得观察的几个点

- 受控 / 非受控组件模式
- Headless / Primitive 组件分层
- CSS-in-JS、原子化 CSS、Design Tokens 等样式路线
- 大型生态下组件 API 如何保持可组合性

## 对 Vue 的启发

你不需要照搬 React 的 API 设计，但可以借鉴它在这些方面的成熟经验：

- 可组合性优先
- 工程约束优先于“表面上更省代码”
- 组件库和应用侧设计系统的职责边界更清晰
