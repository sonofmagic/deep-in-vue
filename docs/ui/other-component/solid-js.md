# SolidJS 组件生态速览

SolidJS 在组件生态上最值得看的，不是“它像不像 Vue”，而是它如何在极细粒度更新模型下组织组件能力。

这让它在一些问题上很适合作为 Vue 的对照样本：

- 如果运行时更轻，组件库要怎样组织交互？
- 如果更新粒度更细，哪些抽象会变得更自然？
- Headless 组件和细粒度响应系统怎样配合？

## 可借鉴点

- 编译期优化思路
- 更小运行时下的交互表达
- Headless 组件在细粒度响应系统中的组合方式

## 对 Vue 的启发

它提醒我们：  
组件库的设计，不只取决于视觉层，也高度取决于底层更新模型和编译策略。
