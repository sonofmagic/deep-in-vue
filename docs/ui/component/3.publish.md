2. **推荐目录结构（例如）：**

   ```
   dist/
     ├── index.esm.js
     ├── index.cjs.js
     ├── index.d.ts
     └── style.css
   ```

3. **配置 package.json 中的入口字段：**

   ```json
   {
     "main": "./dist/index.cjs.js",
     "module": "./dist/index.esm.js",
     "types": "./dist/index.d.ts",
     "style": "./dist/style.css"
   }
   ```
